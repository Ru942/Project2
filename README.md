# 📝 Interactive Character Counter

An elegant and responsive **Character Counter** web application built using **HTML, CSS, and JavaScript**. The project provides real-time character counting with a modern glassmorphism interface, making it both functional and visually appealing.

## 🚀 Live Features

* ✨ Modern glassmorphism UI design
* 📱 Fully responsive layout for desktop, tablet, and mobile devices
* ⌨️ Real-time character counting
* 📊 Animated progress bar indicating character usage
* 🔢 Displays typed and remaining characters instantly
* ✅ Success message when the text is within the character limit
* ⚠️ Warning message when approaching the character limit
* 🚫 Prevents users from exceeding the 200-character limit
* 🗑️ Clear Text button to reset the textarea and counter
* 🎨 Smooth animations and interactive user experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 How It Works

1. Type your message in the textarea.
2. The application updates the character count in real time.
3. A progress bar visually indicates how much of the character limit has been used.
4. A success message appears while the input is within the limit.
5. A warning is displayed when only a few characters remain.
6. Once the maximum limit of **200 characters** is reached, additional input is prevented.
